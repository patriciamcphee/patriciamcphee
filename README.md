## 🚀 What I'm building

### [TechWrit AI](https://techwrit.ai) — Code-aware documentation platform
A SaaS platform that generates structured technical documentation directly from source code, configs, and specs — with style guide enforcement built into the generation layer, not bolted on after.

- 14 documentation modes spanning generation, audit, and AI copilot workflows
- VS Code extension with inline style diagnostics and auto-review on save
- REST API for documentation generation and linting inside CI/CD pipelines
- Live readability engine with Flesch-Kincaid, Flesch Ease, and Gunning Fog scoring

**Stack:** React · TypeScript · REST API · VS Code Extension API · Anthropic

### [UserLens Insights](https://userlens.app) — UX research management platform *(beta)*
A full-featured platform that consolidates participant management, session recording, and synthesis into a single tool — replacing the fragmented combination of spreadsheets, survey tools, and recording software most research teams rely on.

- Anonymous participant tracking with NPS and SUS scoring and historical comparison
- Hypothesis-driven research workflows with structured coverage matrices
- Session recording with screen capture and mouse-click tracking
- Affinity mapping workspace that auto-syncs from participant notes

**Stack:** React · TypeScript · Anthropic
